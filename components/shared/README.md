# Componentes compartidos

Componentes compartidos actuales:

- Header: referencia en `partials/header.html` e inyeccion en `js/layout.js`.
- Footer: `partials/footer.html`, cargado dinamicamente por `js/layout.js`.

Si se migra el header o footer a esta carpeta, actualizar `js/layout.js` y probar todas las paginas publicas.
