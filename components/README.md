# Componentes

Carpeta para ubicar componentes compartidos del sitio.

Actualmente los componentes usados en produccion se conservan en `partials/` para compatibilidad:

- `partials/footer.html`
- `partials/header.html`

`components/shared/` queda como espacio documentado para futuras extracciones sin mezclar componentes con paginas o logica.
