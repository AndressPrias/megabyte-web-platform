# Paginas de tienda

Archivos de entrada de tienda:

| Pagina | Archivo |
| --- | --- |
| Home de tienda | `tienda.html` |
| Detalle de producto | `producto.html` |
| Carrito | `carrito.html` |
| Checkout | `checkout.html` |

Estas paginas usan:

- Logica de tienda desde `js/store.js`.
- Catalogo publico desde `api/products.php`.
- Estilos globales desde `css/styles.css`.
- Imagenes de productos desde `assets/productos/`.
