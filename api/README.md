# API publica

Endpoints publicos:

- `products.php`: devuelve el catalogo de productos para la tienda.
- `tracking.php`: consulta el estado de un servicio por ticket o celular.

Endpoints privados:

- `admin/api.php`: acciones protegidas por sesion de administrador.
- `admin/tracking-api.php`: gestion protegida de tickets de seguimiento.
