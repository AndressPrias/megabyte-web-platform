# Data

Datos locales de respaldo.

- `products.json`: catalogo guardado cuando no hay MySQL disponible.
- `default-products.json`: catalogo base usado para restaurar productos de ejemplo.
- `tickets.json`: tickets de seguimiento cuando no hay MySQL disponible.
- `default-tickets.json`: tickets base de ejemplo para restauracion.

En produccion con MySQL activo, la fuente principal es la tabla `products`.
